🔧 Passos para redefinir a senha do root (modo seguro):
1. Pare o MySQL pelo XAMPP

No Painel do XAMPP, clique em Stop ao lado de "MySQL".

2. Inicie o MySQL em modo seguro (sem verificação de permissões)

Vá até a pasta do XAMPP, geralmente em:

C:\xampp\mysql\bin


Abra um terminal (XAMPP Shell ou CMD) e execute:

mysqld --skip-grant-tables


Isso vai iniciar o MySQL sem verificar usuários e senhas, permitindo acesso irrestrito.

⚠️ Essa janela ficará travada até você encerrar o processo. Deixe ela aberta.

3. Abra outro terminal (novo XAMPP Shell)

No novo terminal, digite:

mysql -u root


Se tudo deu certo, você verá o prompt mysql> — ou seja, você está dentro do MySQL sem precisar de senha.

4. Altere a senha do root

No prompt do MySQL, execute os comandos abaixo (ajuste a senha para o que você quiser):

FLUSH PRIVILEGES;
ALTER USER 'root'@'localhost' IDENTIFIED BY 'nova_senha';


Substitua 'nova_senha' pela nova senha que deseja usar. Exemplo:

ALTER USER 'root'@'localhost' IDENTIFIED BY 'admin123';

5. Feche o terminal do mysqld (Ctrl + C)

Depois de mudar a senha, volte para o terminal onde mysqld --skip-grant-tables estava rodando e pressione Ctrl + C para parar.

6. Reinicie o MySQL normalmente pelo XAMPP

Abra o Painel XAMPP e clique em Start no MySQL.

7. Teste o login novamente

No terminal:

mysql -u root -p


Digite a nova senha (admin123, por exemplo), e você deverá conseguir acesso.
